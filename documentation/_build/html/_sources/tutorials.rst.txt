New import configuration
------------------------

First of all, before configure your import, you must choose a "scope" related to `Nakala roles`_.

.. _Nakala roles: https://documentation.huma-num.fr/nakala/#tableau-recapitulatif-des-droits-par-role

.. image:: images/NakalaImporter_scope.png

An initial call to the API will display Nakala collections and give you the option of importing them by checking a box.
